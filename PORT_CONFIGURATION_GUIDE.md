# 프로젝트 포트 설정 가이드

Remote Chat CLI 서버가 각 프로젝트의 포트 번호를 자동으로 감지하고 상태를 표시할 수 있도록 하는 설정 규칙입니다.

## 포트 감지 우선순위

서버는 다음 순서대로 포트 정보를 검색합니다.

### 1. 명시적 주석 (가장 권장)
코드나 설정 파일에 직접 주석을 달아 포트를 지정할 수 있습니다. 이 방법이 가장 정확하며 우선순위가 높습니다.
- **대상 파일**: `app.py` 또는 `.env`
- **형식**: `# @port: 5000`

### 2. 환경 변수 (`.env`)
프로젝트 루트 폴더의 `.env` 파일에서 다음 변수를 찾습니다.
- `SERVER_PORT=5000`
- `PORT=5000`

### 3. Python 코드 분석 (`app.py`)
`app.py` 파일 내의 실행 코드를 분석하여 포트를 찾습니다.
- `app.run(port=5000)`
- `PORT = 5000`
- `SERVER_PORT = 5000`

---

## 설정 예시

### 방법 A: 주석 사용 (추천)
`app.py` 상단이나 `.env` 파일 아무 곳에나 작성하세요.
```python
# @port: 8080
from flask import Flask
...
```

### 방법 B: .env 파일 사용
```env
PORT=3000
```

### 방법 C: Flask 실행 코드
```python
if __name__ == '__main__':
    app.run(port=5000)
```

## 주의 사항
- 포트 번호는 반드시 **숫자**여야 합니다.
- 주석 사용 시 `@port:` 뒤에 공백이 있어도 상관없습니다 (예: `@port:5000`, `@port: 5000` 모두 가능).
- 여러 설정이 섞여 있을 경우 **주석 > .env > 코드 분석** 순으로 적용됩니다.
